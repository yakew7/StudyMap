# GitHub Student Developer Pack: Step-by-Step Signup

The GitHub Student Developer Pack is the best single thing you can claim as a student in tech. One application gives you 90+ professional tools — IDEs, cloud credits, domains, AI coding assistants, and more — all free while you're enrolled.

## What's included (highlights)

- **GitHub Pro** — private repos, advanced insights, full GitHub features
- **GitHub Copilot Pro** — AI code completion in VS Code, JetBrains, and more
- **JetBrains All Products Pack** — every professional JetBrains IDE for free
- **Canva Pro** — 12 months free
- **Microsoft Azure** — $100 in cloud credits (for students 18+)
- **Namecheap** — free `.me` domain for 1 year + free SSL
- **DigitalOcean** — $200 in cloud hosting credits
- **DataCamp** — 3 months free access to data science courses

The full list is at [education.github.com/pack](https://education.github.com/pack) — it's long.

## Before you start

You'll need:
- A GitHub account (free — create one at [github.com/signup](https://github.com/signup) if you don't have one)
- Either a school-issued email address, **or** a clear photo of your student ID card

School emails ending in `.edu`, `.ac.in`, or your institution's own domain all work. If your school doesn't give email accounts, the ID photo route works fine.

## Step-by-step

**1. Go to the application page**

Visit [education.github.com/pack](https://education.github.com/pack) and click **Get student benefits**.

**2. Sign in**

Log into your GitHub account if you aren't already signed in.

**3. Fill the application form**

- Select your role: **Student**
- Enter your school name and the email address you use there
- Add your expected graduation year
- Write a short note about how you plan to use the tools (one line is fine — e.g. "Building projects for class and learning web development")

**4. Verify your student status**

GitHub will ask you to verify in one of two ways:

- **School email:** If your institution email is recognised, GitHub will send a verification link to it. Click the link and you're done.
- **Student ID upload:** If your email isn't recognised, upload a clear photo of your student ID showing your name, institution name, and validity date. Avoid blurry photos — this is the most common reason for rejection.

**5. Wait for approval**

Most applications are approved within a few hours. Some take up to 7 days if manual review is needed. You'll get an email when it's approved.

**6. Claim individual offers**

Once approved, visit [education.github.com/pack](https://education.github.com/pack) and click **Get access** next to each tool you want. Each tool has its own claim flow — most just link your GitHub account.

## If your application is rejected

Common reasons and fixes:

- **Blurry or cropped ID photo:** Retake it in good lighting, showing the full card.
- **No visible expiry or institution name on ID:** Use your school fee receipt or enrolment letter instead.
- **Personal email used instead of school email:** Add your school email to your GitHub account under Settings → Emails, then reapply.

You can reapply immediately after fixing the issue.

## Renewing every year

The pack expires annually. GitHub will email you a reminder. Just reapply with fresh proof of enrolment — it takes two minutes.
